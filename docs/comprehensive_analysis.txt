# TigerResearchBuddy: Architecture Gap Analysis

## Executive Summary
I have compared our current **"TigerStack 2.0"** implementation against the Detailed Roadmap in `claudesuggestions.txt`.

**Verdict:**
1.  **Phase 1 (Foundation): MATCH ✅**
    *   We built exactly what was suggested: `LanceDB` for vector storage, `Nomic` for embeddings, and `Qwen` for parsing.
    *   *Minor Difference*: Claude suggests using Pydantic for the crawler. Our current JSON-regex approach is working fine, so this is a low-priority refactor.
    *   *Missing*: **Query Expansion**. We should add this (Section 1.3).

2.  **Phase 2 (Graph Layer): MATCH / COMPLETE ✅**
    *   **Implementation**: We built the **"TigerBrain" Hybrid Graph** combining `Site Graph` (Structure) + `Research Graph` (Semantics).
    *   **Architecture**: `GraphBuilder` merges `SmartCrawler` outputs with `DeepDistiller` insights. `GraphRefiner` uses LLMs for taxonomy.
    *   **Visualization**: Successfully visualized ~45k nodes using PyVis (`data/tiger_brain_viz.html`).
    *   **Verdict**: We exceeded the original plan by adding Neural Refinement (Taxonomy/Deduplication).

3.  **Phase 3 (Optimization): FUTURE 🔮**
    *   Reranking and Batch Processing are good next steps once the graph is live.

## Detailed Roadmap Updates

### 1. Immediate Quick Wins (From Clause 1.3)
*   [ ] **Query Rewriter**: Implement the `QueryRewriter` class to expand "ML bias" -> "algorithmic fairness", "machine learning bias". This improves search recall significantly.

### 2. The "Knowledge Graph" Pivot (From Clause 2.1)
*   [ ] **Relation Extractor**: Update `DeepDistiller` to not just extract a list of entities, but **Triples** (`Subject`, `Relation`, `Object`).
*   [ ] **Concept Graph Builder**: Use NetworkX to build the graph of *Concepts*, not just *URLs*.
*   [ ] **Concept Explorer UI**: The Plotly/Streamlit dashboard suggested is superior to our simple PyVis HTML file for data exploration.

### 3. Long Term (Phase 3)
*   [ ] **GraphRAG**: Implementing the "Two-stage retrieval" (Graph traversal + Vector Search) is the end goal.
